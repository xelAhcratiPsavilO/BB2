# Learning Objective

### Set up a Unit Test

In order to setup a Unit Test, it is necessary to:

1) Define RSpec as a testing framework as per
[this](rspec.md).

2) Create a specification file:

```sh
$ tree
.
└── spec
    ├── object.rb
    └── spec_helper.rb
```

3) Describe an object using a specification as per [this](describe.md).
